OxiPulse es un agente de telemetría ultraligero en Rust (Apache 2.0) de SecuryBlack que monitoriza CPU, RAM, Disco y Red vía gRPC/OpenTelemetry.

**Branding:** Metáfora de "pulso vital / constantes vitales" — NO óxido/baremetal.
**Colores:** `#33E1BF` primary, `#0E0E0E` bg negro carbón, `#F0F0F0` texto.
**Relación empresa:** OxiPulse es marca independiente. SecuryBlack aparece solo en footer como "An open-source project by SecuryBlack".
**Dominio:** oxipulse.dev

**Stack web:**
- Next.js 15 App Router + TypeScript
- Tailwind CSS v4
- MDX (next-mdx-remote)
- Framer Motion
- Shiki (syntax highlighting)
